====================
param-tuning-utility
====================
A hyperparameter tuning tool with gorgeous UI for scikit-learn API

============
Requirements
============
param-tuning-utility 0.1.8 requires

* Python >=3.6
* Scikit-learn >=0.24.2
* Numpy >=1.20.3
* Pandas >=1.2.4
* Matplotlib >=3.3.4
* Seaborn >=0.11.0
* Optuna >=2.7.0
* BayesianOptimization >=1.2.0
* MLFlow >=1.17.0
* LightGBM >=3.2.1
* XGBoost >=1.4.2
